# Shared Package V2

下载响应通过 `SharedPackageV2` 提供验证元数据：

| 字段 | 用途 |
| --- | --- |
| `packageVersion` | 固定 `v2` |
| `schemeId`, `parameterSpec`, `proofStatus` | 明示算法边界 |
| `contentKeyVersion` | 密钥版本绑定 |
| `ciphertextHash`, `aadHash`, `capsuleHash` | 材料完整性 |
| `grantPolicyHash`, `grantContextHash` | 授权/上下文绑定 |
| `manifestHash` | 对上述字段的总体摘要 |

`PackageVerifier` 在响应前验证版本、期限、manifest 与 key version。
分块文件可在 manifest 中加入 Merkle root；任意材料替换会在
`PackageVerifierTest` 或 E04 实验中失败。
