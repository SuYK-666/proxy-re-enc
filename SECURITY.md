# Security Policy

ReKeyShare is a security-focused prototype for coursework and research. It is not a production-ready service.

## Supported Versions

Only the current main branch is maintained for experiments and documentation updates.

## Reporting a Vulnerability

If you discover a security issue:

1. Open an issue and prefix the title with "security".
2. Do not include secrets, plaintext, DEK material, private keys, or real user data.
3. Provide minimal reproduction steps, impact assessment, and any relevant logs or traces.

If you need private disclosure, request a private channel in the issue and the maintainer will coordinate the next steps.
